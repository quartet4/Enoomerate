Enoomerate: A Coding Project

A javascript program that lists out all the numbers between a start value and an end value.

Instructions

Enter a starting value on the left and an ending value on the right, then press the "Enoomerate!" button, and the list of values will appear. Note that either value must be between -100 and 100. If the end value is higher than the start value, then the list will count down instead of up. Even values will be bolded.
